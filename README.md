Duck Game - A 2D Multiplayer Game
Duck Game is a 2D multiplayer platformer where players battle it out to be the last duck standing. This project was built from scratch to demonstrate low-level game logic implementation.

This game was made using SFML, applying tile-based rendering and movement mechanics.

Implemented collision detection, map system, and sprite scaling (e.g., 207×144 → 32×32 tiles).
Practiced building struct-based game entities

This project was a collaborative effort develope with a colleagues
